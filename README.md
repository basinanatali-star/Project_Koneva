# Проект "Разработка приложения для виджета банковских операций клиета банка"

## Описание:
Данный проект - это приложение на Python, которое будет готовить данные для отображения в новом виджете

## Установка:
### 1. Клонируйте репозиторий: 

        git clone https://github.com/basinanatali-star/Project_Koneva
### 2. Установите зависимости:

        pip install -r requirements.txt

## Использование:
Это учебный проект, поэтому можно использовать его как образец
